Projects

Dragonfly Synth Module (2023)
This synth module was inspired by the previously designed simpler piano. It uses an AVR MCU and DAC to generate square, sine, triangle, and sawtooth waves. It also features ADSR (attack, decay, sustain, release) envelope control. A separate peripheral board was built to interface with this PCB with 7 potentiometers, 4 LEDs, and a push button.
FSAE Dashboard Electronics (2023)
This PCB was designed with an STM32 microcontroller to process driver controls of a Formula SAE racing car. It fits inside the steering wheel and handles the debouncing of buttons and rotary encoders. It communicates with an LCD dashboard display and other systems via CAN bus.
Piano / Waveform Generator (2023)
This piano runs on a custom GUI that uses a DAC to convert keypresses into an analog voltage. A voltage to frequency converter (VFC) is then used the generate a square wave. A three-stage low pass filter converts the square wave to a sine wave. It also features a volume control and note fading module where a transistor is operated in triode region with non-linear gain.
Ping Pong Ball LED Clock (2022)
This project contains individually addressable LED strips and 128 ping pong balls acting as light diffusers to construct a clock. It was originally inspired by a project on Instructables.com, but the design and codebase were fully customized. It has several modes and animations controlled by an IR remote and Arduino.
Binario AVR Assembly Game (2022)
This board was programmed in AVR assembly as a final project for Caltech EE/CS 10b, Introduction to Digital Logic and Embedded Systems. It is a polled system running on an Atmega64 microcontroller. Features on this boards include a multiplexed 8x8 LED matrix, debounced switches and encoders, sine wave tone generation, and SPI EEROM.
8-bit CPU Design (2022)
A design of an 8-bit CPU was completed on a complex programmable logic device (CPLD) as a final project for Caltech EE/CS 10a. It was programmed in Advanced Boolean Expression Language (ABEL) and it simulates an 8-bit CPU with 85 instructions.
Arduino Smart Home (2021)
This project was designed as 15-lesson course for students to learn about electronics. Each lesson, students learn how to implement and code a new component with an Arduino, including LEDs, buttons, DC motors, servo motors, stepper motors, infrared sensor, liquid crystal displays, joysticks, photoresistors, and ultrasonic sensors.
Maze-Navigating Robot (2023)
The Normstorm is a 2 wheeled autonomous rover designed to explore maps. The bot sees the world through 3 IR sensors and 3 ultrasound detectors. It implements advanced maze navigating algorithms and decision-making processes. The motor and sensors are controlled with a Raspberry Pi.
FIRST Robotics Shooter (2021)
CAD of shooter mechanism that successfully competed in 2020 FIRST Robotics Competition. The mechanism rotates on a turret and has flywheels to launch foam balls into a 3ft wide target from 30 feet away. A robust adjustable hood was also designed to change the launch angle.
